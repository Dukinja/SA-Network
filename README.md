# SA-Network
Multiplayer mod made for GTA:SA using moonloader and SLNet

# Feautres
• OnFoot Sync <br>
• InVehicle Sync <br>
• Health/Death/Respawn Sync

# TODO 
• Weapon, Aim and Shot Sync

# Installation 

Put the files in src into `moonloader` folder. 
Extract `slnet.zip` into `moonloader/lib`
